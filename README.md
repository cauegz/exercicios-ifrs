# Exercícios de Programação - IFRS

Este repositório reúne exercícios, atividades e projetos desenvolvidos ao longo do curso no IFRS, relacionados às disciplinas de programação e áreas próximas da tecnologia.

A ideia deste repositório é centralizar os códigos produzidos durante os estudos, mantendo uma organização geral por disciplina, conteúdo ou período.

Cada pasta pode conter um `README.md` próprio com instruções específicas, explicações, requisitos e detalhes sobre os exercícios daquela disciplina.

## Objetivo

Organizar e documentar os exercícios de programação feitos durante o curso, servindo como:

- registro da evolução nos estudos;
- material de revisão;
- prática de lógica de programação;
- prática com linguagens, ferramentas e conceitos vistos em aula;
- portfólio acadêmico.

## Estrutura do Repositório

Cada pasta de disciplina ou conteúdo possui seu próprio `README.md`, com informações mais específicas sobre:

- objetivo da atividade;
- como executar o código;
- tecnologias utilizadas;
- comandos necessários;
- observações importantes;
- requisitos definidos pelo professor.

## Tecnologias e Conteúdos

Os exercícios deste repositório podem envolver diferentes linguagens, ferramentas e conceitos, como:

- programação orientada a objetos;
- desenvolvimento web;
- banco de dados;
- Java;
- PHP;
- HTML, CSS e JavaScript;
- SQL;
- Git e GitHub.

## Como Usar

Para clonar o repositório:

```bash
git clone https://github.com/cauegz/exercicios-ifrs
```

Depois, acesse a pasta desejada:

```bash
cd nome-da-pasta
```

Em seguida, leia o `README.md` específico da disciplina ou atividade para entender como executar os exercícios.

## Observações

Este repositório tem fins acadêmicos e de estudo.

Alguns exercícios podem representar soluções feitas durante o processo de aprendizagem, portanto podem existir melhorias, refatorações ou diferentes formas de resolver o mesmo problema.

## Status

Repositório em constante atualização conforme novas atividades e exercícios forem desenvolvidos durante o curso.
