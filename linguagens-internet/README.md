# Linguagens para Internet

Esta pasta reúne exercícios, atividades e projetos desenvolvidos na disciplina de **Linguagens para Internet**, com foco no aprendizado de desenvolvimento web.

Durante a disciplina, são trabalhados conceitos relacionados à criação de páginas e sistemas web, utilizando tecnologias como **HTML**, **CSS**, **JavaScript**, **PHP** e integração com **banco de dados**.

A ideia desta pasta é organizar os códigos produzidos durante as aulas, atividades práticas e estudos da disciplina.

## Objetivo

Organizar e documentar os exercícios da disciplina de Linguagens para Internet, servindo como:

- registro da evolução no desenvolvimento web;
- material de revisão para provas e atividades;
- prática com criação de páginas web;
- prática com PHP no lado do servidor;
- prática com formulários, sessões, cookies e banco de dados;
- base para projetos futuros.

## Conteúdos Estudados

Os exercícios desta disciplina podem envolver conteúdos como:

- estrutura básica de páginas HTML;
- estilização com CSS;
- interatividade com JavaScript;
- desenvolvimento back-end com PHP;
- envio e recebimento de dados por formulários;
- métodos `GET` e `POST`;
- uso de sessões;
- uso de cookies;
- conexão com banco de dados;
- autenticação de usuários;
- organização de arquivos em projetos web;
- boas práticas básicas de desenvolvimento.

## Tecnologias Utilizadas

As principais tecnologias utilizadas nesta disciplina são:

- HTML;
- CSS;
- JavaScript;
- PHP;
- SQL;
- MySQL ou MariaDB;
- Git e GitHub.
